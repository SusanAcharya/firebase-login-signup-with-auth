# Flutter Firebase Auth Demo

Flutter + Firebase Auth Demo App that uses Google, Facebook, Email/Password Signup/Login, Email Verification and more!

## Features
- Email/Password Signup
- Email/Password Login
- Email Verification
- Google Sign In
- Facebook Sign In
- Phone Sign In
